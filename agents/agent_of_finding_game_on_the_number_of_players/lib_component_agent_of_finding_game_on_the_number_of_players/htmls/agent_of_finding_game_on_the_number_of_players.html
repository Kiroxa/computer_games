Задачей агента поиска игры от количества игроков является поиск игры, в которую смогут играть одновременно определённое количество игроков. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу поиска игры от количества игроков. Первым аргументом запроса является класс игры. Вторым аргументом запроса является число, представляющее необходимое количество игроков.
